#Arrays in C Programming

This repository contains basic C programs based on arrays
It is part of my learning journey in C programming, focusing on understanding how arrays work and how to use them effectively

# Concepts Covered

* Declaration and initialization of arrays
* Taking array input from user
* Percentage calculation of students using arrays
* Counting occurrences of a character in a string
* Sum and average of array elements
* Finding maximum and minimum values
* Searching elements (linear search)
* Sorting basics

# Programs Included

* Sum of elements in an array
* Average of array elements
* Find largest and smallest element
* Count occurrences of an element
* Reverse an array
* Linear search program
* Simple sorting (ascending/descending)

# Purpose

The goal of this repository is to strengthen my understanding of arrays in C and improve problem-solving skills through practice.
